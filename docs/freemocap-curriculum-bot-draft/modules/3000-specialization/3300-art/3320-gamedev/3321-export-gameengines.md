# 3321 - Export for Game Engines

```{note}
This module covers preparing and exporting motion capture data for use in game engines.
```

## Learning Objectives

- Understand game engine animation requirements
- Optimize animations for real-time playback
- Configure FBX export settings for games
- Prepare animation clips for game use

## Topics Covered

- Real-time vs offline animation differences
- FBX format for game engines
- Coordinate system conversions (Y-up vs Z-up)
- Scale and unit considerations
- Animation clip preparation
- Keyframe reduction for performance
- Root motion setup
- Animation loop preparation

## Prerequisites

- Completed [3320 - Video Game Design Overview](./3320-gamedev-overview.md)
- Basic Blender export familiarity

## Deliverable

**Game-ready animation export**

Export a FreeMoCap animation for game engine use:
- Proper FBX settings
- Optimized keyframe count
- Documentation of export settings

**Bot chat** discussing export considerations for different engines.
