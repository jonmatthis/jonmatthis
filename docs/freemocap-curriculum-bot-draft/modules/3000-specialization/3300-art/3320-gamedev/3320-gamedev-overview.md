# 3320 - Video Game Design: Overview

```{note}
This sub-path focuses on using motion capture for video game character animation and interactive applications.
```

## Learning Objectives

- Understand game engine requirements for animation
- Learn real-time animation optimization techniques
- Integrate motion capture into game projects
- Create playable characters driven by mocap

## Sub-Path Modules

| Module | Title | Focus |
|--------|-------|-------|
| 3320 | Overview | Game development concepts |
| 3321 | Export for Game Engines | FBX preparation, optimization |
| 3322 | Godot Integration | Godot animation system, import |
| 3323 | Unreal/Unity Integration | Major engine workflows |
| 3324 | Real-Time Applications | VTubing, live performance |

## Prerequisites

- Completed [3300 - Art Track Overview](../3300-art-overview.md)
- Basic understanding of game development concepts
- Familiarity with at least one game engine helpful
