# 3322 - Godot Integration

```{note}
This module covers integrating motion capture animations into the Godot game engine.
```

## Learning Objectives

- Import mocap animations into Godot
- Work with Godot's AnimationPlayer and AnimationTree
- Set up character controllers with mocap
- Create blended animation states

## Topics Covered

- Godot scene structure for characters
- Importing FBX/GLTF animations
- AnimationPlayer setup
- AnimationTree state machines
- Animation blending
- Root motion in Godot
- Character controller basics
- Triggering animations from code

## Prerequisites

- Completed [3321 - Export for Game Engines](./3321-export-gameengines.md)
- Basic Godot familiarity (GDScript or C#)

## Deliverable

**Playable Godot character**

Create a Godot project with:
- Imported mocap animation
- Basic character controller
- Animation state machine

**Bot chat** discussing Godot animation workflow.
