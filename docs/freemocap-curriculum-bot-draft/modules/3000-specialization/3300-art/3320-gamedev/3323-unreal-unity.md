# 3323 - Unreal & Unity Integration

```{note}
This module covers integrating motion capture animations into Unreal Engine and Unity.
```

## Learning Objectives

- Import mocap animations into Unreal and Unity
- Work with each engine's animation systems
- Set up character blueprints/prefabs
- Understand engine-specific workflows

## Topics Covered

### Unreal Engine
- Animation import and retargeting
- Animation Blueprints
- Control Rig basics
- Skeletal mesh setup

### Unity
- Mecanim animation system
- Avatar configuration
- Animator Controller setup
- Animation events

### Comparison
- Workflow differences
- Performance considerations
- When to choose which engine

## Prerequisites

- Completed [3322 - Godot Integration](./3322-godot-integration.md)
- Basic familiarity with Unreal or Unity

## Deliverable

**Playable character in Unreal OR Unity**

Create a project in your engine of choice with:
- Imported mocap animation
- Basic character setup
- Animation state management

**Bot chat** comparing engine approaches.
