# 3310 - Animation: Overview

```{note}
This sub-path focuses on using motion capture for character animation in Blender and other 3D software.
```

## Learning Objectives

- Understand animation workflows using motion capture
- Learn to clean and polish mocap data
- Master character rigging and retargeting
- Create production-quality animated content

## Sub-Path Modules

| Module | Title | Focus |
|--------|-------|-------|
| 3310 | Overview | Animation concepts and workflows |
| 3311 | Mocap Cleanup | Polishing raw motion capture data |
| 3312 | Character Rigging | Creating animation-ready character rigs |
| 3313 | Retargeting | Transferring motion to different characters |
| 3314 | Production Workflows | Rendering, compositing, final output |

## Prerequisites

- Completed [3300 - Art Track Overview](../3300-art-overview.md)
- Basic Blender familiarity
- Understanding of FreeMoCap output data
