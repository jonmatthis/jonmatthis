# 3311 - Mocap Cleanup & Polish

```{note}
This module covers techniques for cleaning and polishing raw motion capture data for animation.
```

## Learning Objectives

- Identify and fix common mocap artifacts
- Use Blender's animation tools for cleanup
- Apply smoothing without losing character
- Prepare mocap data for production use

## Topics Covered

- Common mocap problems (jitter, drift, pops)
- Graph Editor workflow in Blender
- Keyframe reduction techniques
- Smoothing and filtering in Blender
- Fixing foot sliding and contact issues
- Ground plane alignment
- Manual keyframe adjustment
- Preserving motion character while cleaning

## Prerequisites

- Completed [3310 - Animation Overview](./3310-animation-overview.md)
- Basic Blender animation tools familiarity

## Deliverable

**Cleaned animation file**

Take raw FreeMoCap output and create a cleaned version:
- Document issues identified
- Show before/after comparison
- Export cleaned animation

**Bot chat** discussing cleanup strategies.
