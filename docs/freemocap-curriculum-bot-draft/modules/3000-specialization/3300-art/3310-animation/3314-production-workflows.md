# 3314 - Production Workflows

```{note}
This module covers end-to-end production workflows from motion capture to final rendered output.
```

## Learning Objectives

- Plan and execute mocap-driven animation projects
- Integrate mocap with keyframe animation
- Set up scenes for rendering
- Create polished final output

## Topics Covered

- Project planning for mocap animation
- Combining mocap with hand-keyed animation
- Multi-take management and library organization
- Lighting and camera for animated scenes
- Render settings and optimization
- Compositing basics
- Output formats and delivery
- Collaboration workflows

## Prerequisites

- Completed [3313 - Motion Retargeting](./3313-retargeting.md)

## Deliverable

**Short animated sequence**

Create a short (10-30 second) animated piece:
- Motion capture driven character
- Basic scene setup
- Rendered output (image sequence or video)
- Documentation of production process

**Bot chat** reflecting on your production workflow.
