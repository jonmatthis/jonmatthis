# 3300 - Art Track: Overview

```{note}
This track focuses on artistic applications of motion capture, including character animation and video game development.
```

## Learning Objectives

- Understand motion capture workflows for artistic applications
- Learn to apply mocap data to character animation
- Explore game engine integration
- Create compelling artistic works using FreeMoCap data

## Art Sub-Paths

Choose one or more sub-paths based on your interests:

| Sub-Path | Modules | Focus |
|----------|---------|-------|
| [331x: Animation](./3310-animation/3310-animation-overview.md) | 3310-3314 | Character animation, Blender workflows, retargeting |
| [332x: Video Game Design](./3320-gamedev/3320-gamedev-overview.md) | 3320-3324 | Game engines, real-time characters, interactive applications |

## Animation vs Game Development

**Choose Animation (331x) if you're interested in:**
- Film and video production
- Pre-rendered content
- Cinematic quality output
- Blender-focused workflows

**Choose Video Game Design (332x) if you're interested in:**
- Real-time applications
- Interactive experiences
- Game engine workflows (Godot, Unity, Unreal)
- VTubing and live performance

## Prerequisites

- Completed [3000 - Specialization Overview](../3000-overview.md)
- Interest in digital art, animation, or game development
- Basic familiarity with 3D software helpful
