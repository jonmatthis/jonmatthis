# 3142 - Armatures & Animation

```{note}
This module covers armature creation and animation data handling in Blender.
```

## Learning Objectives

- Understand Blender armature structure
- Create bone hierarchies programmatically
- Work with animation data (F-curves, keyframes)
- Apply motion capture data to armatures

## Topics Covered

- Armature data structure
- Edit mode vs pose mode
- Bone relationships and parenting
- Bone constraints
- Animation curves (F-curves)
- Keyframe insertion and manipulation
- Motion capture data to Blender animation
- T-pose and rest pose considerations

## Prerequisites

- Completed [3141 - Blender Python API](./3141-blender-api.md)

## Deliverable

**Armature creation script**

Create a Python script that:
- Builds an armature programmatically
- Adds animation data
- Demonstrates bone hierarchy concepts

**Bot chat** discussing armature and animation concepts.
