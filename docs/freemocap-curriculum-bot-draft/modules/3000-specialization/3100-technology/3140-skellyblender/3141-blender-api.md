# 3141 - Blender Python API

```{note}
This module covers the Blender Python API fundamentals needed for addon development.
```

## Learning Objectives

- Understand Blender's Python scripting environment
- Learn bpy module structure and conventions
- Create and manipulate Blender objects programmatically
- Develop basic addon structure

## Topics Covered

- bpy module overview
- Data blocks and object types
- Creating meshes, armatures, empties
- Operators and panels
- Addon registration
- Blender's undo system
- Script vs addon development

## Prerequisites

- Completed [3140 - SkellyBlender Overview](./3140-skellyblender-overview.md)
- Python programming experience
- Basic Blender navigation

## Deliverable

**Simple Blender script**

Create a Python script in Blender that:
- Creates objects programmatically
- Manipulates object properties
- Demonstrates understanding of bpy structure

**Bot chat** discussing Blender scripting patterns.
