# 3143 - Export Pipelines

```{note}
This module covers exporting motion capture data from Blender to various formats.
```

## Learning Objectives

- Understand common animation export formats
- Configure FBX export for different targets
- Work with BVH motion capture format
- Create custom export workflows

## Topics Covered

- FBX format and options
- BVH format structure
- Export settings for game engines
- Export settings for other 3D software
- Coordinate system conversions
- Scale and unit considerations
- Batch export workflows

## Prerequisites

- Completed [3142 - Armatures & Animation](./3142-armatures.md)

## Deliverable

**Export workflow documentation**

Document an export pipeline for a specific target:
- Configure export settings
- Test in target application
- Document any necessary adjustments
