# 3140 - SkellyBlender: Overview

```{note}
SkellyBlender (FreeMoCap Blender Addon) provides Blender integration for visualizing and working with motion capture data.
```

## Learning Objectives

- Understand the FreeMoCap Blender addon architecture
- Learn about armature creation and animation data
- Explore export formats (FBX, BVH)
- Understand Blender Python API basics

## What is SkellyBlender?

The FreeMoCap Blender Addon provides:
- Automatic scene setup from FreeMoCap data
- Armature generation from skeletal data
- Animation curve creation
- Export to standard formats (FBX, BVH)
- Visualization tools
