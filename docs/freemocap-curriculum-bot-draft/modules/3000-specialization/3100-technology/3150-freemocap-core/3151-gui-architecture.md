# 3151 - GUI Architecture

```{note}
This module covers FreeMoCap's Qt-based graphical user interface architecture.
```

## Learning Objectives

- Understand Qt/PySide application structure
- Learn about FreeMoCap's widget organization
- Explore signal/slot communication patterns
- Understand the main window and view management

## Topics Covered

- PySide6/PyQt fundamentals
- Widget hierarchy in FreeMoCap
- Signal and slot connections
- Main window organization
- View switching and navigation
- Settings and preferences UI
- Custom widget development

## Prerequisites

- Completed [3150 - FreeMoCap Core Overview](./3150-core-overview.md)
- Python GUI development helpful

## Deliverable

**GUI component analysis**

Document the structure of a specific FreeMoCap GUI component:
- Widget hierarchy
- Signal/slot connections
- User interaction flow

**Bot chat** discussing Qt patterns in FreeMoCap.
