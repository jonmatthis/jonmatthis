# 3150 - FreeMoCap Core: Overview

```{note}
FreeMoCap Core is the main application that orchestrates the entire motion capture pipeline.
```

## Learning Objectives

- Understand the FreeMoCap main application architecture
- Learn about the Qt-based GUI structure
- Explore the pipeline orchestration logic
- Understand the recording session data model

## What is FreeMoCap Core?

The main FreeMoCap application provides:
- Qt-based graphical user interface
- Pipeline orchestration (recording → processing → visualization)
- Session and recording management
- Integration of all sub-skelly components
- User-facing configuration and controls
