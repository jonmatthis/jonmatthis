# 3132 - Filtering & Interpolation

```{note}
This module covers signal processing techniques for cleaning motion capture data.
```

## Learning Objectives

- Understand filtering techniques for motion data
- Learn interpolation methods for gap-filling
- Apply appropriate parameters for different movements
- Evaluate filter effects on data quality

## Topics Covered

- Butterworth low-pass filtering
- Filter order and cutoff frequency selection
- Gap detection in time series
- Linear and spline interpolation
- Handling extended gaps
- Before/after comparison techniques
- Parameter selection guidelines

## Prerequisites

- Completed [3131 - Multi-View Triangulation](./3131-triangulation.md)
- Basic signal processing concepts

## Deliverable

**Filtering comparison report**

Process the same data with different filter settings:
- Compare multiple cutoff frequencies
- Document effects on different body parts
- Identify optimal parameters for your data
- Visualize before/after results

**Bot chat** discussing filter parameter selection.
