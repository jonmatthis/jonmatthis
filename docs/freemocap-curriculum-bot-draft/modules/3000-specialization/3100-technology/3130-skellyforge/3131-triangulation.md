# 3131 - Multi-View Triangulation

```{note}
This module covers the mathematics and implementation of 3D reconstruction from multiple camera views.
```

## Learning Objectives

- Understand multi-view geometry fundamentals
- Learn triangulation algorithms
- Work with camera matrices and calibration data
- Analyze reprojection error

## Topics Covered

- Epipolar geometry basics
- Camera projection matrices
- Direct Linear Transform (DLT) triangulation
- Reprojection error calculation
- Handling outliers and noise
- Anipose-based calibration integration
- Coordinate system transformations

## Prerequisites

- Completed [3130 - SkellyForge Overview](./3130-skellyforge-overview.md)
- Linear algebra fundamentals
- Matrix operations with NumPy

## Deliverable

**Triangulation analysis**

Analyze a calibrated recording:
- Extract camera matrices
- Manually triangulate sample points
- Calculate and interpret reprojection errors
- Document the reconstruction quality
