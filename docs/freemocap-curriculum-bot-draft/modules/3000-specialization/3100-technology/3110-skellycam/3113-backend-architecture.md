# 3113 - Backend Architecture

```{note}
This module covers SkellyCam's backend architecture including FastAPI, WebSockets, and multi-process design.
```

## Learning Objectives

- Understand SkellyCam's FastAPI/Uvicorn server architecture
- Learn about WebSocket communication for real-time video
- Explore multi-process camera management
- Understand the React/Electron frontend integration

## Topics Covered

- FastAPI application structure
- Uvicorn ASGI server
- WebSocket streaming for live video preview
- Multi-process architecture for camera handling
- Inter-process communication patterns
- Frontend-backend communication
- API endpoint design

## Prerequisites

- Completed [3112 - Video Synchronization](./3112-synchronization.md)
- Familiarity with async Python (asyncio)
- Basic web development concepts

## Deliverable

**Architecture documentation**

Create an annotated diagram showing:
- Server components and their interactions
- Data flow from cameras to frontend
- WebSocket communication patterns

**Bot chat** demonstrating understanding of the backend architecture.
