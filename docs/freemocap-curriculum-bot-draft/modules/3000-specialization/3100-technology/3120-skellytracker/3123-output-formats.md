# 3123 - Output Formats & Data Structures

```{note}
This module covers SkellyTracker's output data formats and coordinate systems.
```

## Learning Objectives

- Understand SkellyTracker's output data structures
- Learn about keypoint coordinate systems
- Work with confidence scores and visibility flags
- Parse and transform tracker output

## Topics Covered

- Keypoint data structure (x, y, confidence)
- Image coordinate systems
- Normalized vs pixel coordinates
- Confidence and visibility interpretation
- Output array shapes and indexing
- Converting between formats
- Handling missing/occluded keypoints

## Prerequisites

- Completed [3122 - Pose Estimation Models](./3122-pose-models.md)
- NumPy familiarity

## Deliverable

**Data extraction script**

Write a script that:
- Loads raw SkellyTracker output
- Extracts specific body parts
- Transforms coordinates as needed
- Handles missing data appropriately

**Bot chat** demonstrating understanding of output formats.
