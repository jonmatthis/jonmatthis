# 3210 - Biomechanics: Overview

```{note}
This sub-specialization focuses on biomechanics - the study of mechanical laws relating to movement and structure of living organisms.
```

## Learning Objectives

- Understand fundamental biomechanics concepts
- Apply biomechanical analysis to motion capture data
- Analyze human movement patterns scientifically
