# 3202 - Intro to Data Analysis

```{note}
This module introduces data analysis concepts and techniques for motion capture research.
```

## Learning Objectives

- Understand data analysis workflows for motion capture

- Apply filtering and smoothing techniques

## Topics Covered

- Data analysis fundamentals
- Time series analysis basics
- Filtering and smoothing
- Data visualization
- Statistical analysis introduction
- Working with NumPy and Pandas
