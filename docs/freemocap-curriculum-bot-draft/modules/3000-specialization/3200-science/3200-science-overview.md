# 3200 - Science Track: Overview

```{note}
This track focuses on scientific applications of motion capture, including measurement, data analysis, biomechanics, and neuroscience.
```

## Learning Objectives

- Understand scientific approaches to motion capture data
- Learn fundamental measurement and analysis concepts
- Choose a sub-specialization in biomechanics or neuroscience
