# 2004 - Data Output Details

```{note}
This module provides detailed understanding of FreeMoCap's data output formats and how to work with them.
```

## Learning Objectives

- Understand all FreeMoCap output file formats in depth
- Navigate and locate specific data within output folders
- Modify and customize Jupyter notebooks for analysis
- Extract and manipulate motion capture data programmatically

## Topics Covered

- Output folder structure and file organization
- NumPy array formats and dimensions
- CSV formats (tall vs. wide)
- Coordinate systems and conventions
- Data validation and quality assessment
- Custom analysis workflows

## Prerequisites

- Completed [2003 - Mocap Environment Optimization](./2003-environment-optimization.md)

## Deliverable

**Multiple deliverables:**

1. **Quiz** - Demonstrate knowledge of data output locations and formats
2. **Tweaked Jupyter notebook** - Modified notebook showing custom analysis
3. **Bot chat** - Interactive discussion demonstrating data understanding
