# Static Images

Place your curriculum images here:

- `freemocap-logo.svg` - Main logo
- `favicon.ico` - Browser favicon
- `skelly-mascot.png` - Skelly mascot image
- `mocap-overview.svg` - Motion capture overview diagram
- `distortion-types.svg` - Lens distortion examples

## Recommended Image Formats

- **SVG** for diagrams and logos (scales perfectly)
- **PNG** for screenshots and complex images
- **WebP** for photos (MyST auto-converts)

## Tips

- Keep images under 500KB when possible
- Use descriptive filenames
- Include alt text in your markdown for accessibility
